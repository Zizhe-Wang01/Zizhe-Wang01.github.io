## 关于这个网站

这个网站不是一份已经完成的教材，而是一套持续生长的个人知识库。

我会尽量把每个问题写成：

1. 我最初不明白什么；
2. 核心概念是什么；
3. 为什么要这样设计；
4. 一个可以运行或验证的例子；
5. 容易混淆的地方。
